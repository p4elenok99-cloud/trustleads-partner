# TrustLeads — partner.trust-leads.ru

Третий сайт проекта: партнёрский лендинг в едином стиле с `web/` и `web-academy/`.
Один самодостаточный `index.html` (HTML/CSS/JS без сборки), деплой через GitHub Pages из ветки `main`.

Контент воспроизводит структуру основного сайта trust-leads.ru (Tilda), но собран как чистый код:
- герой «Рост начинается с партнёрства»;
- секция «О нас» + карточки 01–03 (Экосистема / Философия / Присоединяйтесь);
- «Преимущества сотрудничества» (6 карточек) и «Что Вы получаете» (6 карточек);
- блок «Сотрудничество» с кнопками Telegram/MAX и формой заявки;
- футер с контактами (телефон, e-mail, режим работы, адрес).

## Настройки (в `index.html` → `CONFIG`)

| Поле | Назначение |
|------|------------|
| `telegramUrl`, `maxUrl` | Мессенджеры. Те же ссылки, что на основном сайте TrustLeads. |
| `phone`, `phoneHref`, `email` | Контакты (телефон рисуется `tel:`, почта — `mailto:`). |
| `privacyUrl` | Ссылка/файл политики конфиденциальности. Пусто — ссылка неактивна. |
| `formEndpoint` | URL Google Apps Script для приёма заявок. Пусто — приём выключен. |
| `metrikaId` | Номер счётчика Яндекс.Метрики (только цифры). Пусто — аналитика выключена. |

## Форма заявки → Google Sheets

1. Создайте новую Google-таблицу (`https://sheets.new`).
2. «Расширения» → «Apps Script», вставьте код из `google-apps-script.gs`, сохраните.
3. «Развернуть» → «Новое развёртывание» → «Веб-приложение»: запуск «от имени меня»,
   доступ «Все». Скопируйте URL `/exec`.
4. Вставьте URL в `CONFIG.formEndpoint`.

Колонки таблицы: `Дата | Имя | Фамилия | Телефон | Телеграм`.

## Деплой

`git add . && commit && push` в репозиторий сайта → включить GitHub Pages (ветка `main`) →
в настройках Pages указать домен `partner.trust-leads.ru` (файл `CNAME` уже в папке) →
у DNS-провайдера добавить запись `CNAME partner → <аккаунт>.github.io`.

## Локальный просмотр

```
python -m http.server 5520 --directory web-partner
```
(конфиг превью `partner` уже добавлен в `.claude/launch.json`).
